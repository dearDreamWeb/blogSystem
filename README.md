# 论坛功能
采用了响应式布局，移动端也可以使用。
可以实现登录和注册，在论坛发帖，点赞，评论，修改个人资料，搜索文章等功能。

# 核心技术栈
- vue
- express
- mysql
- element-ui
- scss
- es6

# 灵感来源
论坛名字是自己想的，叫做 天方夜谈，logo和favicon都是自己做的。
像布局之类的大多数是借鉴了csdn和知乎这两大论坛，少部分借鉴的是微博。
